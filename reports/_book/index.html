Single-step Hybrid Predition

Single-step Hybrid Predition

Matthias Westhues, Claas Heuer

2016-09-14

Preface

This book covers an extension of the methodology of single-step prediction — which was developed for the imputation of SNP data from pedigree information — to the imputation of quantitative predictor variables using genomic information.

The companion R-package for this analysis is sspredr. The development version can be installed from Github:

devtools::install_github("mwesthues/sspredr")
